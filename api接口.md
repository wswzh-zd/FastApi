## 1.接口设计规范 RESTful

| 请求方法 | 请求地址 | 后端操作 |
| :--- | :--- | :--- |
| POST | `/student/` | 增加学生 |
| GET | `/student/` | 获取所有学生 |
| GET | `/student/1` | 获取id为1的学生 |
| PUT | `/student/1` | 修改id为1的学生 |
| DELETE | `/student/1` | 删除id为1的学生 |

restful规范是一种通用的规范，不限制语言和开发框架的使用。事实上，我们可以使用任何一门语言，任何一个框架都可以实现符合restful规范的API接口。
